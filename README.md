# Gerador de Relatórios a partir de CSV

Projeto em desenvolvimento para leitura, processamento e geração de relatórios com base em arquivos CSV.

## Funcionalidades
- Leitura de arquivos CSV
- Filtragem de dados por cliente
- Geração de relatórios automatizados

## Tecnologias utilizadas
- Python
- Streamlit

## Status
🚧 Em desenvolvimento

## Objetivo
Praticar manipulação de dados, lógica de programação e automação de processos.
